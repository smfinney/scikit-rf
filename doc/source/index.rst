**scikit-rf** (aka `skrf`) is an Open Source, BSD-licensed package for RF/Microwave engineering developed and maintained for all `supported versions <https://www.python.org/downloads/>`_  of the Python programming language (currently 3.6+). It provides a modern, object-oriented library which is both flexible and scalable. The documentation below is broken up into three sections; narrative tutorials,  practical examples, and a reference API.
